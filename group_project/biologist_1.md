**Paper 1: Spike Interface**

- Paper: [eLife](https://elifesciences.org/articles/61834)  
- Notes: The eLife paper provides analysis notebooks that reproduce 
figures  

**Paper 2: Protein Function Prediction Method**  
- Code: [GitHub](https://github.com/yuxjiang/CAFA2)  
- Paper: [Genome 
Biology](https://genomebiology.biomedcentral.com/articles/10.1186/s13059-016-1037-6)  
- Notes: GitHub provides the step by step for building baseline predictors  

**Paper 3: Bactopia**  
- Paper: [PMC](https://pmc.ncbi.nlm.nih.gov/articles/PMC7406220/)  
- Data and Code: [GitHub](https://github.com/bactopia/bactopia/)  
- Notes: Contains the raw sequences and links to all the software used in 
the study as well as a step by step guide on GitHub  

**Paper 4: Generalized leaky integrate-and-fire models classify multiple 
neuron types**  
- Paper: [PubMed](https://pubmed.ncbi.nlm.nih.gov/29459723/)  
- Data: [Cell Types](http://celltypes.brain-map.org)  
- Analysis code: 
[GitHub](https://github.com/AllenInstitute/GLIF_Teeter_et_al_2018)  
- Code for pink noise stimulus: 
[GitHub](https://github.com/AllenInstitute/ephys_pink_noise)  
- Notes: Paper seems promising as the data is easily accessible and the 
GitHub provides text files of source code and step by step instructions on 
which codes to run in which order  

**Paper 5: Predictive Big Analytics: A study of Parkinson's Disease Using 
Large, Complex, Heterogenous, Incongruent, Multi-Source and Incomplete 
Observations**  
- Paper: [PLOS 
ONE](https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0157077#sec025)  
- Raw data: [PPMI](http://www.ppmi-info.org/data)  
- Code: [GitHub](https://github.com/BD2K/BDDS)  
- Notes: This paper involves an archive of data on Parkinson's disease 
including neuroimaging, genetics, clinical, and demographic data. Their 
GitHub provides the complete Predictive Big Data Analytics protocol to 
obtain vector of neuroimaging biomarkers and R-script for machine learning 
diagnostic classification
